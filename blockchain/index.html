Blockchain

A numbered Tetris like Game

Rules

The goal of the game is not to be submerged and go for the high score.

The firewall trigger once every 7 moves and add a line to the bottom.

To eliminate numbers, a number have to be equal to sum of numbers or firewalls in the same row or column and in the same stack.

The game is available to play @ https://echkourine25.github.io/blockchain11.html

The Github Page is @ https://github.com/echkourine25/blockchain/
